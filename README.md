# Oil Reserves and Production 3D Map
An interactive 3D visualization built with Mapbox GL JS, showing global oil reserves (height) and oil production (color) by country.

## Features
- 3D extruded polygons representing oil reserves
- Blue color gradient showing oil production levels
- Hover tooltips with detailed values
